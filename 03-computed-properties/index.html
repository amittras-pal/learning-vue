Computed Properties.

They're like values resolved using React.useMemo.

PROBLEM::Using methods to resolve values in the template.

- WHY?

Because every time some data changes, Vue has to execute the method to check if something changed in that section. Overriding the rendering optimizations.

- HOW TO RESOLVE??

  1. Avoid things like this.
  2. Use computed properties.

Example:

Count Value: {{count}}

Entered Value using a method: {{outputFullName()}}

Entered Value using computed Value: {{fullName}}
